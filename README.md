# rand_num_generator
 A random number generator for Android devices. <br>
 To deal with the constant trouble caused by my life partner ;), when choosing couple of random numbers for the monthly lottery.<br>
 This app is written as a simple user interface, where the users can decide the amount and range of the random numbers, which they want to generate.<br>
 Only available for Android devices, as there are only Android Phones in my home.<br>
 Project fulfills the basic needs of finding numbers for the lottery but is still on developing.<br>
